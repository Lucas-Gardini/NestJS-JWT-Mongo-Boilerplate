## Descrição

[NestJS](https://github.com/nestjs/nest) Boilerplate com autenticação JWT e conexão banco de dados MONGODB

## Instalação

```bash
$ yarn install
```

## Running the app

```bash
# development
$ yarn start

# watch mode
$ yarn start:dev

# build
$ yarn build:windows
#ou
$ yarn build:linux

# production mode
$ yarn start:prod
```
